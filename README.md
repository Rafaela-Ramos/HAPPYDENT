# Doc Smile Suite - Frontend

## Información del proyecto

Aplicación frontend construida con Vite, React, TypeScript, Tailwind CSS y shadcn/ui.

## Requisitos

- Node.js y npm

## Scripts

```sh
# Instalar dependencias
npm i

# Entorno de desarrollo
npm run dev

# Compilar para producción
npm run build

# Previsualización de la build
npm run preview
```

## Tecnologías

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Estructura

- `src/`: código fuente de la aplicación
- `public/`: recursos estáticos

## Notas

- Configura variables de entorno y endpoints según tus necesidades.
- Consulta `package.json` para ver todas las dependencias y scripts disponibles.
